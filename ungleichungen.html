Kapitel 7 Ungleichungen

7.1 Definition von Ungleichungen

Definition 7.1 (Ungleichungen) Definition (Ungleichungen)

Verbindet man zwei Zahlen/ Ausdrücke durch eines der Vergleichssymbole \(<,\leq,>\) oder \(\geq\) so entsteht eine Aussage, die in Abhängigkeit von den Zahlen/ Auswertung der Ausdrücke entweder wahr oder falsch ist:

  • \(a\text{ < }b\) (gesprochen: \(a\) kleiner \(b\)) ist wahr, wenn die Zahl \(a\) kleiner und nicht gleich \(b\) ist.

  • \(a\leq b\) (gesprochen: \(a\) kleiner gleich \(b\)) ist wahr, wenn die Zahl \(a\) kleiner oder gleich \(b\) ist.

  • \(a> b\) (gesprochen: \(a\) größer \(b\)) ist wahr, wenn die Zahl \(a\) größer und nicht gleich \(b\) ist.

  • \(a\geq b\) (gesprochen: \(a\) größer gleich \(b\)) ist wahr, wenn die Zahl \(a\) größer oder gleich \(b\) ist.

Die Vergleichszeichen drücken auf dem Zahlenstrahl aus, wie die gegebenen Werte zueinander liegen: \(a\text{ < }b\) bedeutet, dass \(a\) links von \(b\) auf dem Zahlenstrahl liegt. Dabei ist \(a\text{ < }b\) gleichbedeutend mit \(b>a\), sowie \(a\leq b\) gleichbedeutend mit \(b\geq a\) ist.

Treten Terme mit Unbestimmten in einer Ungleichung auf, so besteht die Aufgabe darin, den Zahlenbereich für die Unbestimmte zu ermitteln, so dass die Ungleichung wahr ist.

Beispiel 7.1 (Ungleichungen) Beispiel (Ungleichungen)

Die Ungleichung \(2x-1>0\) ist wahr für \(x>\frac 12\).

7.2 Umformung von Ungleichungen

Die einfachen linearen Umformungen aus dem Abschnitt Gleichungen sind auch für Ungleichungen zulässig. Sie verändern die Lösungsmenge der betrachteten Ungleichung nicht. Zu beachten ist, dass Multiplikation einer Ungleichung mit negativer Zahl die Richtung der Ungleichung verändert.

Definition 7.2 (Umformen von Ungleichungen) Regeln (Umformen von Ungleichungen)

Multipliziert man eine Ungleichung mit einem Term, der eine Variable \(x\) enthält, so muss man eine Fallunterscheidung führen:

  • Für diejenigen \(x\), für die der multiplizierte Term positiv ist, bleibt die Richtung der Ungleichung erhalten.

  • Für diejenigen \(x\), für die der multiplizierte Term negativ ist, wird das Vergleichssymbol umgedreht.

  • Der Fall, dass der multiplizierte Term den Wert Null annimmt, muss bei der Umformung ausgeschlossen und ggf. separat betrachtet werden.

Die Addition von Termen, welche die Variable enthalten, erfordert dagegen keine Fallunterscheidung. Umformungen mit Fallunterscheidungen sind meist notwendig, wenn die Unbestimmte im Nenner oder in zusammengesetzten Termen auftritt:

Beispiel 7.2 (Ungleichungen) Beispiel (Ungleichungen)

Die Ungleichung \(\frac 1{2x}\leq 1\) kann vereinfacht werden, indem beide Seiten der Ungleichung mit dem Term \(2x\) multipliziert werden:

  • Unter Bedingung \(2x>0\) \((x>0)\) erhält man die neue Ungleichung \(1\leq 2x\) mit der Lösungsmenge \(x\geq \frac 12\) oder anders geschrieben \(L_1=[\frac 12; \infty)\).

  • Unter Bedingung \(2x< 0\) \((x< 0)\) erhält man die neue Ungleichung \(1\geq 2x\), die gilt, wenn \(x\leq \frac 12\). Wegen der zusätzlichen Bedingung \(x< 0\) ist die Lösungsmenge \(x< 0\) oder anders geschrieben \(L_2=(-\infty; 0)\).

  • Im Fall \(x=0\) darf die Multiplikation mit \(2x\) nicht durchgeführt werden. Dieser Wert ist keine Lösung.

Insgesamt erhält man also die Vereinigungsmenge \(L = L_1 \cup L_2 = (-\infty; 0) \cup [\frac 12; \infty)\) oder anders audgedrückt: \(L=\{x\in \mathbb R: x\text{ < } 0 \text{ oder } x\geq \frac 12\}\).

In jedem Fall prüft man und ggfs. reduziert man die erhaltene Lösungsmenge auf die Teilmenge, die der Fallbedingung genügt. Die Vereinigung der angepassten Lösungsmengen der einzelnen Fälle bildet die Lösungsmenge der ursprünglichen Ungleichung.

7.3 Quiz Ungleichungen

7.4 Spezielle Ungleichungen\(^\ast\)

7.4.1 Faktorisierte Ungleichungen

Definition 7.3 (Faktorisierte Ungleichungen) Definition (Faktorisierte Ungleichungen)

Hat man eine Ungleichung, in der auf einer Seite \(>0\) bzw. \(\text{ < }0\) steht und die andere Seite in faktorisierter Form (mit höchstens zwei Faktoren) gebenen ist, so kann man sie lösen, in dem folgende Fallunterscheidungen vornimmt.

Ungleichung mit \(>0\) bzw. \(\geq 0\) gilt:

  • wenn beide Faktoren einzeln \(>0\) bzw. \(\geq 0\) sind,
  • wenn beide Faktoren einzeln \(\text{ < }0\) bzw. \(\leq 0\) sind.

Ungleichung mit \(\text{ < }0\) bzw. \(\leq 0\) gilt: - wenn einer der Faktoren \(>0\) bzw. \(\geq 0\) ist und der andere \(\text{ < }0\) bzw. \(\leq 0\) ist.

Beispiel 7.3 (Faktorisierte Ungleichungen) Beispiel (Faktorisierte Ungleichungen)

Die Ungleichung \((x-1)(2x+1)>0\) gilt:

  • Unter Bedingung \(x-1>0\) und \(2x+1>0\), also für Werte \(x>1\) und
  • unter Bedingung \(x-1\text{ < }0\) und \(2x+1 \text{ < }0\), also für Werte \(x\text{ < }-\frac 12\).

Insgesamt erhält man also die Lösungsmenge \(L=\{x\in \mathbb R: x\text{ < }-\frac 12 \text{ oder } x> 1\}\).

7.4.2 Betragsungleichungen

Analog zum Vorgehen in Gleichungen löst man Beträge in Ungleichungen durch eine Fallunterscheidung auf.

Definition 7.4 (Betragsungleichungen) Rechenregel (Betragsungleichungen)

Eine Ungleichung mit einem Betragsausdruck wird in zwei Fälle unterteilt:

  • Für diejenigen \(x\), für die der Term im Betrag nicht negativ ist, kann der Betrag weggelassen bzw. durch einfache Klammern ersetzt werden.
  • Für diejenigen \(x\), für die der Term im Betrag negativ ist, wird der Term in Klammern gesetzt und mit einem Minus davor versehen (negiert).

Die Lösungsmengen aus den Fällen werden wie im Fall der Multiplikation mit \(x\)-Term eingeschränkt und zur Lösungsmenge für die ursprüngliche Ungleichung vereinigt.

Beispiel 7.3 (Betragsungleichungen) Beispiel (Betragsungleichungen)

Die Ungleichung \(|4x-2|< 1\) teilt man in zwei Fälle auf:

  • Für \(4x-2\geq0\) ( \(x\geq\frac 12\) ) erhält man \((4x-2)\text{ < }1\) bzw. \(x\text{ < }\frac 34\). Wegen der Fallbedingung \(x\geq\frac 12\) ist die Lösungsmenge \(\{x\in\mathbb R: \frac 12\leq x\text{ < }\frac 34\}\) oder anders geschrieben \(L_1=[\frac 12; \frac 34)\).

  • Für \(4x-2\text{ < }0\) ( \(x\text{ < }\frac 12\) ) erhält man die \(-(4x-2)\text{ < }1\) bzw. \(x>\frac 14\). Wegen der Fallbedingung \(x\text{ < }\frac 12\) ist die Lösungsmenge \(\{x\in\mathbb R: \frac 14\text{ < } x\text{ < }\frac 12\}\) oder anders geschrieben \(L_2=(\frac 14; \frac 12)\).

Insgesamt erhält man also die Vereinigungsmenge \(L = L_2\cup L_1 =(\frac 14; \frac 12) \cup [\frac 12; \frac 34) = (\frac 14; \frac 34)\) oder anders audgedrückt: \(L=\{x\in \mathbb R:\frac 14\text{ < } x\text{ < } \frac 34\}\).
