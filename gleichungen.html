Vor- und Brückenkurs: Wirtschaftsmathematik

Kapitel 6 Gleichungen

Gleichungen beschreiben Zusammenhänge zwischen Ausdrücken mit unbekannten Größen oder stellen ein zu lösendes Problem dar.

6.1 Definition einer Gleichung

Definition 6.1 (Gleichung) Definition (Gleichung)

Eine Gleichung ist ein mathematischer Ausdruck der Form:

Linke Seite = Rechte Seite

wo in der Regel mindestens eine Variable ( \(x\) ) vorkommt.

Eine Gleichung wird durch manche Variablenwerte erfüllt und durch andere Werte nicht. Um dies zu prüfen, müssen die Werte in die Gleichung eingesetzt werden. Die Gleichung wird durch die eingesetzten Werte erfüllt, falls die beiden Seiten der Gleichung übereinstimmen:

Beispiel 6.1 (Gleichung) Beispiel (eine Gleichung)

Die Gleichung \(2x-1=x^2\) besitzt die rechte Seite \(x^2\) und die linke Seite \(2x-1\). Einsetzen von \(x=1\) ergibt den Wert \(1\) auf beiden Seiten, also ist \(x=1\) eine Lösung dieser Gleichung. Dagegen ist \(x=2\) keine Lösung, denn eine Auswertung ergibt \(4\) auf der linken und \(3\) auf der rechten Seite der Gleichung.

6.2 Lösen von Gleichungen

Eine Gleichung zu lösen bedeutet alle Zahlen zu finden, die, wenn sie an der Stelle der Variablen \(x\) in die Gleichung eingesetzt sind, beide Seiten angleichen.

Definition 6.2 (Lösung einer Gleichung) Definition (Lösungsmenge)

Die Lösungsmenge \(L\) einer Gleichung ist die Menge aller Zahlen, die, wenn in die Gleichung an Stelle der Variablen eingesetzt, dazu führen, dass die Gleichung erfüllt ist. Die Lösungsmenge wird in der Mengennotation angegeben (bspw. \(L=\{1;2;4\}\) ).

Typische Lösungsschritte für Gleichungen sind:

  • das Finden einer Gleichung, die ein textuell gegebenes Problem beschreibt.
  • Umformen der Gleichung, insb. Auflösen nach den Variablen.
  • Angabe der Lösungen einer Gleichung, d.h. aller Werte für die Unbestimmten, welche die Gleichung erfüllen.

Beispiel 6.2 (Wertpapierthändlerin Tina) Beispiel (Wertpapierthändlerin Tina)

Tina handelt gern mit Wertpapieren in ihrem Online-Wertpapierdepot. Sie tätigt eine Transaktion zum Kauf von Wertpapieren und bezahlt insgesamt \(\color{blue}{100}\) Euro. Für jede Transaktion muss sie den festen Preis von \(\color{red}1\) Euro und den variablen Preis von \(\color{green}{10\%}\) (oder \(\color{green}{0,1}\)) des Transaktionsvolumen an die Bank zahlen. Wie viel Geld hat nun Tina in die Wertpapiere investiert?

Antwort

Die Aufgabenstellung wird zunächst in eine Gleichung übersetzt, wobei die Variable \(x\) den investierten Betrag (den Transaktionsvolumen) darstellt. Die Gleichung lautet dann \(\color{red}1 + \color{green}{0,1} x + x = \color{blue}{100}\). Sie drückt aus, dass der gezahlte Betrag (rechte Seite) die festen und die variablen Kosten der Transaktion und die eigentliche Investition in die Wertpapiere decken soll (linke Seite).

Diese Gleichung kann dann nach \(x\) aufgelöst werden, in dem man von beiden Seiten \(1\) abzieht und dann durch \(1,1\) teilt.

Die Lösung ist \(x=90\), Tina hat also eine Investition von 90 Euro in ihr Wertpapiergeschäft getätigt.

Zum Umformen der Gleichungen sind folgende Operationen erlaubt:

  • Addieren/Subtrahieren von Termen auf beiden Seiten der Gleichung,
  • Vertauschung der beiden Seiten der Gleichung,
  • Umformen von Termen auf einer Seite der Gleichung,
  • bekannte Zusammenhänge einsetzen,
  • Multiplikation/Division mit einem Term (dieser Term darf nicht Null sein),
  • Bilden des Kehrwerts auf beiden Seiten der Gleichung (beide Seiten müssen verschieden von Null sein).

Solche Umformungen verändern nicht die Lösungsmenge einer Gleichung, man spricht von äquivalenten Gleichungen in dem Fall und drückt dies durch das Symbol “\(\Leftrightarrow\)” aus.

Beispiel 6.3 (Umformen von Gleichungen)

  • Umformen von \(12+t=\frac{2t}{2t^2}+t.\)

\[\begin{align} &12+t=\frac{2t}{2t^2}+t, ~~| -t,\\ &\Leftrightarrow 12=\frac{2t}{2t^2}, ~~|\text{Gleichung umgedreht},\\ &\Leftrightarrow \frac{2t}{2t^2}=12, ~~|: 2t,\\ &\Leftrightarrow \frac 1t = 12, ~~| \text{Kehrwerte genommen},\\ &\Leftrightarrow t = \frac 1{12}. \end{align}\]

  • Umformen von \(\frac{-18 -2(2y-3)}{3-(4y-3)}=4.\)

\[\begin{align} &\frac{-18 -2(2y-3)}{3-(4y-3)}=4,\\ &\Leftrightarrow \frac{-18 -4y+6}{3-4y+3}=4, ~~|\text{Klammer aufgemacht},\\ &\Leftrightarrow \frac{-12 -4y}{6-4y}=4, ~~|\text{Terme zusammengefasst},\\ &\Leftrightarrow \frac{-12 -4y}{6-4y}\cdot (6-4y)=4\cdot (6-4y), ~~|\cdot (6-4y),\text{ beachte }y\not=-\frac32,\\ &\Leftrightarrow -12 -4y=24-16y, ~~| +4y-24,\\ &\Leftrightarrow -36 = -12y~~| :(-12),\\ &\Leftrightarrow 3 = y. \end{align}\]

6.4 Spezielle Gleichungen

6.4.1 Faktorisierte Gleichungen

Definition 6.3 (Faktorisierte Gleichung) Definition (Faktorisierte Gleichung)

Ist eine Gleichung, in der die rechte Seite gleich null ist, in faktorisierter Form gebenen oder lässt sie sich in Faktoren zerlegen, so kann man sie lösen, in dem man jeden Faktor gleich Null setzt.

Zwei Bedingungen müssen also erfüllt sein:

  • auf der rechten Seite der Gleichung stehe \(0\)
  • die linke Seite der Gleichung besteht aus Faktoren (bspw. \((x-1)(x+2)=0\))

Beispiel 6.4 (Faktorisieren von Gleichungen) Beispiel (Lösung einer faktorisieren Gleichung)

Die Gleichung \((2x+1)(0,5x-2) = 0\) hat die Lösungsmenge \(L=\{-\frac{1}{2};4\}\).

Die Lösungen erhält man in dem man den ersten Faktor \(2x+1\) gleich Null setzt (ergibt \(x=-\frac 12\)) und den zweiten Faktor \(0,5-2\) dann auch gleich Null setzt (ergibt \(x=4\)).

Wird in einer faktorisierten Gleichung noch zusätzlich durch einen Term geteilt (z.b. \(\frac{(x-1)(x-2)}{x^2 -4} = 0\)), so setzt man den Zähler gleich Null und prüft zusätzlich, dass die Lösung der Definitionsmenge angehört (Nenner ungleich Null).

Beispiel 6.5 (Faktorisieren von Gleichungen) Beispiel (Lösung einer faktorisieren Gleichung mit Bruchtermen)

Der Zähler der Gleichung \(\frac{(x-1)(x-2)}{x^2 -4} = 0\) ist gleich Null für \(x\in\{1;~2\}\). Als Definitionsmenge ermittelt man \(x^2 -4\not=0\) \(\rightarrow x\not\in\{-2;~2\}\) oder \(\mathbb D=\mathbb R\setminus\{-2;~2\}\). Also ist die Lösungsmenge \(L=\mathbb D\cap\{1;~2\} =\{1\}\).

6.4.2 Betragsgleichungen\(^\ast\)

Definition 6.4 (Gleichungen mit Betrag) Definition (Betragsgleichungen)

Die Betragsfunktion \(|x|\) ordnet jedem reelen \(x\) seinen Wert ohne das Vorzeichen zu:

  • ist \(x\geq 0\), so ist \(|x|=x,\)
  • ist \(x< 0\), so ist \(|x|=-x ,\) als Ergebnis hat man dann eine positive Zahl, da sich hinter \(x\) eine negative Zahl verbirgt.

Betragsgleichungen sind Gleichungen, in denen ein oder mehrere Beträge vorkommen. Diese sind problematisch, da der Betrag eines Terms letztlich über eine Fallunterscheidung:

\[\begin{align} |\text{Term}| = \begin{cases}\text{Term, falls Term}\geq 0,\\ -\text{Term, falls Term}< 0. \end{cases} \end{align}\]

Beispiel 6.6 (Lösen von Betragsgleichungen) Beispiel (Lösen von Betragsgleichungen)

Die Betragsgleichung \(|x|=2\) hat die Lösungsmenge \(L=\{2;-2\}\).

Die Betragsgleichung \(|2x-1|=5\) hat die Lösungsmenge \(L=\{3;-2\}\) resultierend aus:

  • \(2x-1=5,\) falls \(2x-1\geq 0\) ( \(x\geq\frac 12\) ),
  • \(1-2x=5,\) falls \(2x-1\text{ < }0\) ( \(x\text{ < }\frac 12\) ).
