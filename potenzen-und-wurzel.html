Kapitel 3 Potenzen und Wurzel

3.1 Potenzen

Potenzen mit natürlichem Exponenten werden folgendermaßen definiert:

Definition 3.1 (Potenz) Definition (Potenzieren) Die \(n\)-te Potenz ( \(n\in \mathbb N\) ) einer Zahl \(a\in \mathbb R\) ist das \(n\)-fache Produkt der Zahl \(a\) mit sich selbst: \[a^n = a\cdot a\cdot a \cdot \ldots \cdot a.\]

\(a\) wird als Basis und \(n\) als Exponent bezeichnet.

Dabei gibt es einige besondere Fälle, die man am besten auswendig können sollte:

  • \(a^0 = 1\) auch \(0^0=1\)
  • Für \(n>0\): \(0^n=0\)
  • \((-1)^n=\begin{cases}-1, n\text{ ungerade},\\ ~~~1, n\text{ gerade}\end{cases}\)

Definition 3.2 (negative Potenz) Definition (negative Potenzen) Potenzen mit negativen ganzzahligen Exponenten werden definiert durch die Formel \(a^{-n} = \frac 1{a^n},~n\in\mathbb N, a\not=0.\)

Folglich berechnet sich \(2^{-2}=\frac 1{2^2}=\frac 14\). Analog ergibt sich \((-2)^{-3}=\frac1{(-2)^3}=-\frac18.\)

Definition 3.3 (Rechenregel für Potenzen) Definition (Rechenregel für Potenzen) Folgende Rechenregeln gelten:

\[a^n a^m = a^{n+m}\]

\[ (a^{n})^{m} = a^{nm}\]

\[\frac{a^n}{a^m} = a^{n-m}\]

\[\frac{a^n}{a^n}= a^0=1\]

Außerdem gelten: \(a^n\cdot b^n=(a\, b)^n; a^n/b^n=(a/b)^n\) und \(a^{-n}=1/a^n\).

Beispiel 3.1 (Potenzieren) Beispiel (Potenzieren)

\[3^4\cdot 3^2 = 3^{4+2} = 3^6\] \[(3^{4})^{2} = 3^{4\cdot 2} = 3^8\] \[\frac{3^{4}}{3^{2}} = 3^{4-2} = 3^2\] \[3^4\cdot 4^4 = (3\cdot 4)^{4} = 12^4\] \[\frac{3^4}{4^4} = \left(\frac 34\right)^{4}\]

3.2 Wurzel

Bei einem rationalen Exponenten der Form \(\frac 1n, n\in\mathbb N\) muss die Definition wieder erweitert werden, um zum Beispiel \(4^{\frac 12}\) berechnen zu können. Diese Potenz lässt sich auch in Wurzelschreibweise umwandeln und man erhält \(4^{\frac 12}=\sqrt[2]{4}=\sqrt{4}=2.\) Allgemein gilt \(\sqrt[n]{a} = a^{\frac1n}\).

Dies führt auf die Umkehrung der Potenzrechnung, das Wurzelziehen.

Definition 3.4 (Wurzel) Definition (Wurzel)

Die \(n\)-te Wurzel einer Zahl \(a\geq 0\), ist diejenige Zahl, deren \(n\)-te Potenz gleich \(a\) ist:

\[a^{\frac 1n} = \sqrt[n]{a} =b \Rightarrow b^n=a.\]

\(a\) wird als Wurzelbasis und \(n\) als Wurzelexponent bezeichnet.

D.h. \[16^{\frac 12} =\sqrt[2]{16}=4; ~~ 27^{-\frac13}=\frac1{27^{\frac 13}}= \frac1{\sqrt[3]{27}} = \frac 13.\]

Definition 3.5 (Rechenregeln für Wurzel) Definition (Rechenregeln für Wurzel) Für \(n,m\in\mathbb N, a,b \in \mathbb R_+\) gelten die folgenden Wurzelrechenregeln:

  • Zwei Wurzeln mit gleichem Exponenten werden multipliziert, indem die Wurzel aus dem Produkt der Wurzelbasen gezogen wird, der Wurzelexponent bleibt unverändert: \[\sqrt[n]{a}\cdot\sqrt[n]{b}=\sqrt[n]{a\cdot b}.\]

  • Zwei Wurzeln mit gleichem Exponenten werden dividiert, indem die Wurzel aus dem Quotienten der Wurzelbasis gezogen wird, der Wurzelexponent bleibt unverändert: \[\frac{\sqrt[n]{a}}{\sqrt[n]{b}}=\sqrt[n]{\frac a b}.\]

  • Die \(m\)-te Potenz einer Wurzel bildet man, indem die \(m\)-te Potenz der Wurzelbasis nimmt, der Wurzelexponent bleibt unverändert: \[(\sqrt[n]{a})^m = \sqrt[n]{a^m} = a ^{\frac mn}.\]

  • Die \(m\)-te Wurzel aus einer Wurzel bildet man, indem die Wurzelexponenten multipliziert werden und die Basis gleich bleibt: \[\sqrt[m]{\sqrt[n]{a}} = \sqrt[m\cdot n]{a} = a^{\frac 1 {mn}}.\]

Beispiel 3.2 (Wurzel) Beispiel (Wurzel) \[\sqrt[3]{3}\cdot\sqrt[3]{9} = \sqrt[3]{3\cdot 9} = \sqrt[3]{27}=3.\] \[\frac{\sqrt[4]{32}}{\sqrt[4]{2}} = \sqrt[4]{\frac{32}2} = \sqrt[4]{16}=2.\] \[\sqrt[10]{4^5} = (4^{5})^{\frac 1{10}} = 4^{\frac 5{10}} = 4^{\frac 1{2}}=\sqrt[2]{4}=2.\] \[\sqrt[2]{\sqrt[3]{x}}=\sqrt[2\cdot 3]{x}=\sqrt[6]{x}=x^{\frac 16}.\]
